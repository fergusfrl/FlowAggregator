# Flow Aggregator

> Restful API which aggregates up to date New Zealnd river data

## Quick Start

```bash
# Install dependencies
npm install

# Run Flow Aggregator
npm start

# Server runs on http://localhost:3030
```

## Requests

```bash
# GET
http://localhost:3030
```
